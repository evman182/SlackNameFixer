Name Fixer for Slack

If your company uses SSO for Slack, your preferred name may get overwritten every time you log in. Use Name Fixer to set your preferred name. It will monitor your profile for name changes, and then set it back to the preferred name you set.

Commands:

/set_preferred_name [name] - Set your preferred full name. Any changes to your full name will trigger the app to set your full name back to the preferred name you set.

/get_preferred_name - Get the preferred name you previously set.

/unset_preferred_name - Unset your preferred name setting. The app will no longer modify your profile when it detects a name change.
